# WoM Helper

[![EN](https://img.shields.io/badge/EN-gray)](../README.md) [![ZH-CN](https://img.shields.io/badge/ZH--CN-gray)](./README.zh-CN.md) [![JA](https://img.shields.io/badge/JA-gray)](./README.ja.md) ![RU](https://img.shields.io/badge/RU-blue) [![DeepWiki](https://deepwiki.com/badge.svg)](https://deepwiki.com/fzlins/WoM-Helper)

Пользовательский скрипт Tampermonkey для [minesweeper.online](https://minesweeper.online/), улучшающий отображение конфигурации поля, добавляющий режим «Без флагов», показывающий прогноз очков события на странице событий, включающий автоматический поиск соперника на странице PvP, добавляющий быстрые однокнопочные действия на странице квестов и позволяющий включать и отключать каждую функцию на странице настроек.

## Настройки

На [странице настроек](https://minesweeper.online/settings) в нижней части панели настроек появляется раздел **WoM Helper**. Используйте его для включения или отключения следующих функций; изменения вступают в силу при следующем переходе на страницу этой функции — перезагрузка не требуется:

- Ссылки на поля и плотность мин
- Прогноз очков события
- Получение всех наград за квесты
- Автопрокрутка к своей позиции

## Возможности

- **Ссылки на поля и плотность мин** — Любая спецификация поля в формате `WxH/M` (например, `30x16/99`) в любом месте страницы автоматически превращается в кликабельную ссылку, запускающую игру с этой конфигурацией. Рядом с каждой спецификацией отображается процент плотности мин, чтобы с первого взгляда оценить сложность.

- **Переключатель NF (Без флагов)** — На игровых страницах рядом с выбором сложности появляется флажок **NF** (работает и на компьютере, и на мобильном). При включении правый клик на поле блокируется, предотвращая случайную установку флагов — отлично для практики игры без флагов. Настройка запоминается между сессиями.
- **Колонка прогноза события** — На [странице событий](https://minesweeper.online/events) в рейтинге появляется колонка 🎯 с прогнозируемым итогом очков каждого игрока к концу события, исходя из текущего темпа. Наведите на любую ячейку, чтобы увидеть дневное среднее.
- **Автопрокрутка к своей позиции** — Когда рейтинговая таблица загружается или ваша позиция меняется после обновления статистики, страница автоматически прокручивается к вашей строке в таблице. Смена страниц таблицы этого не вызывает.
- **Автопоиск соперника (PvP)** — На [странице PvP](https://minesweeper.online/pvp) рядом с кнопкой *Найти соперника* появляется флажок **Auto**. При включении кнопка нажимается автоматически, как только становится доступной — при загрузке страницы, после каждого матча или таймаута. Нажмите *Отмена*, чтобы остановить. Настройка запоминается между сессиями.
- **Получение всех наград за квесты** — На [странице квестов](https://minesweeper.online/quests) вверху каждой категории квестов появляется кнопка *Получить всё*, если есть доступные награды. Один клик — и все доступные награды в этой категории получены; кнопка исчезает, как только все забраны.

## Установка

**Вариант А — Greasy Fork (рекомендуется)**

Установите напрямую с [Greasy Fork](https://greasyfork.org/scripts/578042-minesweeper-online-helper).

**Вариант Б — Вручную**

1. Установите браузерное расширение [Tampermonkey](https://www.tampermonkey.net/).
2. Откройте [wom-helper.user.js](https://raw.githubusercontent.com/fzlins/WoM-Helper/main/wom-helper.user.js) в браузере.
3. Tampermonkey предложит установить скрипт — нажмите **Установить**.

> **Скрипт не работает?** В браузерах на основе Chrome Tampermonkey 5.3+ требует однократной дополнительной настройки для запуска пользовательских скриптов. Подробнее см. [Q209: Разрешение на выполнение пользовательских скриптов](https://www.tampermonkey.net/faq.php?q=Q209#Q209):
> - **Chrome/Edge 138+** — Правой кнопкой мыши нажмите на значок Tampermonkey → *Управление расширением* → включите **Разрешить пользовательские скрипты**.
> - **Старые версии Chrome/Edge** — Перейдите по адресу `chrome://extensions` (или `edge://extensions`) и включите **Режим разработчика**.
