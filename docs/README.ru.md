# WoM Helper

[English](../README.md) [简体中文](./README.zh-CN.md) [日本語](./README.ja.md) Русский [DeepWiki](https://deepwiki.com/fzlins/WoM-Helper)

Пользовательский скрипт Tampermonkey для [minesweeper.online](https://minesweeper.online/), улучшающий отображение конфигурации поля, добавляющий режим «Без флагов», показывающий прогноз очков события на странице событий и включающий автоматический поиск соперника на странице PvP.

## Возможности

- **Кликабельные ссылки поля** — Любой текст в формате `WxH/M` автоматически преобразуется в кликабельную ссылку, запускающую игру с заданной конфигурацией.
- **Плотность мин** — После каждой спецификации поля отображается процент плотности мин.
- **Языковые ссылки** — Ссылки генерируются с учётом языкового префикса текущей страницы.
- **Переключатель NF (Без флагов)** — Вспомогательная функция для игроков, практикующих игру без флагов. На страницах игры после селектора уровня *Custom* добавляется флажок **NF** (в строке для настольных компьютеров и в мобильном выпадающем меню). При включении все правые клики на игровом поле блокируются, предотвращая случайную установку флагов. Настройка сохраняется между сессиями через `localStorage`.
- **Колонка прогноза события** — На [странице событий](https://minesweeper.online/events) в таблицу рейтинга добавляется колонка 🎯 с прогнозируемым итогом очков каждого игрока к концу события на основе текущего темпа. При наведении на ячейку отображается среднее значение `/d` (в день).
- **Автоклик по рейтингу** — При загрузке страницы, если `#stat_my_rank` содержит действительный номер позиции, скрипт автоматически кликает по ссылке рейтинга один раз. При изменении значения рейтинга (например, после обновления статистики) клик повторяется. Смена страниц таблицы не вызывает этого действия.
- **Автопоиск соперника (PvP)** — На [странице PvP](https://minesweeper.online/pvp) рядом с кнопкой *Найти соперника* добавляется флажок **Auto**. При включении кнопка автоматически нажимается каждый раз, когда становится доступной (при загрузке страницы и после каждого матча или таймаута). Нажатие кнопки *Отмена* снимает флажок и останавливает цикл. Настройка сохраняется между сессиями через `localStorage` и работает на всех языковых версиях страницы.
- **Получение всех наград за квесты** — На [странице квестов](https://minesweeper.online/quests) в ячейке заголовка последнего столбца каждой таблицы квестов добавляется кнопка (повторяющая стиль и текст кнопки получения награды на сайте), если в таблице есть доступные для получения строки. Нажатие на неё одновременно получает все доступные награды в этой таблице. Кнопка исчезает автоматически, как только все награды получены.
- **Динамический контент** — Использует `MutationObserver` для обработки контента, загружаемого после первоначального рендеринга страницы.

## Установка

**Вариант А — Greasy Fork (рекомендуется)**

Установите напрямую с [Greasy Fork](https://greasyfork.org/scripts/578042-minesweeper-online-helper).

**Вариант Б — Вручную**

1. Установите браузерное расширение [Tampermonkey](https://www.tampermonkey.net/).
2. Откройте [wom-helper.user.js](https://raw.githubusercontent.com/fzlins/WoM-Helper/main/wom-helper.user.js) в браузере.
3. Tampermonkey предложит установить скрипт — нажмите **Установить**.

> **Скрипт не работает?** В браузерах на основе Chrome Tampermonkey 5.3+ требует однократной дополнительной настройки для запуска пользовательских скриптов. Подробнее см. [Q209: Разрешение на выполнение пользовательских скриптов](https://www.tampermonkey.net/faq.php?q=Q209#Q209):
> - **Chrome/Edge 138+** — Правой кнопкой мыши нажмите на значок Tampermonkey → *Управление расширением* → включите **Разрешить пользовательские скрипты**.
> - **Старые версии Chrome/Edge** — Перейдите по адресу `chrome://extensions` (или `edge://extensions`) и включите **Режим разработчика**.
