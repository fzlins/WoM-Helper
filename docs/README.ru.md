# WoM Helper

[English](../README.md) [简体中文](./README.zh-CN.md) [日本語](./README.ja.md) Русский [DeepWiki](https://deepwiki.com/fzlins/WoM-Helper)

Пользовательский скрипт Tampermonkey для [minesweeper.online](https://minesweeper.online/), улучшающий отображение конфигурации поля и добавляющий режим «Без флагов».

## Возможности

- **Кликабельные ссылки поля** — Любой текст в формате `WxH/M` (например, `58x35/393`) автоматически преобразуется в кликабельную ссылку, запускающую игру с заданной конфигурацией.
- **Плотность мин** — После каждой спецификации поля отображается процент плотности мин, например `58x35/393 (19.36%)`.
- **Языковые ссылки** — Ссылки генерируются с учётом языкового префикса текущей страницы (например, `/cn/start/58x35/393` для китайских страниц, `/start/58x35/393` для английских).
- **Переключатель NF (Без флагов)** — Вспомогательная функция для игроков, практикующих игру без флагов. На страницах игры после селектора уровня *Custom* добавляется флажок **NF** (в строке для настольных компьютеров и в мобильном выпадающем меню). При включении все правые клики на игровом поле блокируются, предотвращая случайную установку флагов. Настройка сохраняется между сессиями через `localStorage`.
- **Динамический контент** — Использует `MutationObserver` для обработки контента, загружаемого после первоначального рендеринга страницы.

## Установка

**Вариант А — Greasy Fork (рекомендуется)**

Установите напрямую с [Greasy Fork](https://greasyfork.org/scripts/578042-minesweeper-online-helper).

**Вариант Б — Вручную**

1. Установите браузерное расширение [Tampermonkey](https://www.tampermonkey.net/).
2. Откройте панель управления Tampermonkey и нажмите **Создать новый скрипт**.
3. Замените содержимое по умолчанию на [wom-helper.user.js](../wom-helper.user.js).
4. Сохраните. Скрипт будет автоматически активироваться на `https://minesweeper.online/*`.

## Формула плотности

$$\text{плотность} = \frac{\text{мины}}{W \times H} \times 100\%$$

Пример: `58x35/393` → $393 \div (58 \times 35) \approx 19.36\%$
