# WoM Helper

[English](../README.md) [简体中文](./README.zh-CN.md) 日本語 [Русский](./README.ru.md) [DeepWiki](https://deepwiki.com/fzlins/WoM-Helper)

[minesweeper.online](https://minesweeper.online/) 向けの Tampermonkey ユーザースクリプトです。ボード設定の表示を強化し、ノーフラッグモードを追加し、イベントページにイベントスコア予測を表示し、PvP ページに対戦相手自動検索機能を追加し、クエストページにワンクリックショートカットを提供します。

## 機能

- **ボードリンクと地雷密度** — ページ上のどこにある `WxH/M` 形式のボード仕様（例: `30x16/99`）も自動的にクリック可能なリンクに変換され、クリックするとその構成でゲームが展開されます。各仕様の後には地雷密度のパーセンテージも表示され、難易度を一目で判断できます。
- **NF（ノーフラッグ）トグル** — ゲームページの難易度セレクター横に **NF** チェックボックスが表示されます（デスクトップ・モバイル両対応）。チェックを入れるとボード上の右クリックが無効になり、誤ってフラグを置くことを防ぎます — ノーフラグスタイルの練習に最適です。設定はセッションをまたいで記憦されます。
- **イベント統計列** — [イベントページ](https://minesweeper.online/events)のランキングに 🎯 列が表示され、現在のペースに基づいて各プレイヤーのイベント終了時の予測合計ポイントを確認できます。任意のセルにカーソルを当てると日平均が表示されます。
- **ランク自動スクロール** — ランキングの読み込み時や統計更新後に順位が変わった際、ページが自動的にランキング内の自分の位置までスクロールします。ページング操作では発動しません。
- **対戦相手自動検索（PvP）** — [PvP ページ](https://minesweeper.online/pvp)の *対戦相手を探す* ボタンの隣に **Auto** チェックボックスが表示されます。チェックを入れると、ページ読み込み時・各試合後・タイムアウト後にボタンが自動でクリックされます。*キャンセル* をクリックすると停止します。設定はセッションをまたいで記憦され、すべての言語版ページで動作します。
- **クエスト報酬一括受け取り** — [クエストページ](https://minesweeper.online/quests)で受け取れる報酬がある場合、各クエストカテゴリーの上部に *まとめて受け取る* ボタンが表示されます。ワンクリックでそのカテゴリーの全ての受け取り可能な報酬を受け取れ、全て受け取るとボタンは自動的に消えます。

## インストール

**方法 A — Greasy Fork（推奨）**

[Greasy Fork](https://greasyfork.org/scripts/578042-minesweeper-online-helper) から直接インストールしてください。

**方法 B — 手動インストール**

1. [Tampermonkey](https://www.tampermonkey.net/) ブラウザ拡張機能をインストールしてください。
2. ブラウザで [wom-helper.user.js](https://raw.githubusercontent.com/fzlins/WoM-Helper/main/wom-helper.user.js) を開きます。
3. Tampermonkey がインストールを促すダイアログを表示しますので、**インストール** をクリックしてください。

> **スクリプトが動作しない場合** Chrome 系ブラウザで Tampermonkey 5.3+ を使用する場合、ユーザースクリプトを実行するために一度だけ追加設定が必要です。詳細は [Q209: ユーザースクリプトの実行許可](https://www.tampermonkey.net/faq.php?q=Q209#Q209) を参照してください：
> - **Chrome/Edge 138+** — Tampermonkey アイコンを右クリック → *拡張機能を管理* → **ユーザースクリプトを許可** を有効化。
> - **旧バージョンの Chrome/Edge** — `chrome://extensions`（または `edge://extensions`）を開き、**デベロッパー モード** を有効化。

## 密度の計算式

$$\text{密度} = \frac{\text{地雷数}}{W \times H} \times 100\%$$
